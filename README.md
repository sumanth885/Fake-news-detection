# Fake News Detection

## 📌 Description
A simple machine learning project that detects whether news is real or fake using Python.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Naive Bayes

## ▶️ How to Run

1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python fake_news.py

3. Enter any news text to test prediction

## 📊 Output
- Displays model accuracy
- Predicts whether input news is Real or Fake
